# Power Distribution Analysis and Auto Theft Detection IoT Grid

## Overview

This project presents an IoT-enabled smart monitoring system capable of analyzing electrical parameters while simultaneously detecting vehicle theft attempts. The system continuously monitors voltage and current using dedicated sensors and provides live data visualization through the Blynk IoT platform. A magnetic reed switch detects unauthorized access and instantly triggers theft alerts.

## Features

- Real-time Voltage Monitoring
- Real-time Current Monitoring
- Live Blynk Dashboard
- Auto Theft Detection
- Instant Alert Notifications
- Cloud-based Monitoring
- Low-cost IoT Architecture

## Hardware Used

- ESP32
- ACS712 Current Sensor
- ZMPT101B Voltage Sensor
- Magnetic Reed Switch
- Wi-Fi Network
- Blynk IoT Platform

## Software

- Arduino IDE
- Blynk IoT
- ESP32 Libraries

## Working

1. ESP32 reads voltage and current.
2. Values are processed and transmitted over Wi-Fi.
3. Live readings are displayed on the Blynk dashboard.
4. Reed switch continuously monitors unauthorized access.
5. Theft events generate immediate alerts.
